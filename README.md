# CourseApply Australia Landing Page

## Find Your VET College in 60 Seconds

A high-performance, single-page landing page designed for EduPath Australia to streamline the VET college application process.

## 🚀 Features
- **Mobile-First Design**: Fully responsive across all devices.
- **Micro-Animations**: Clean hover effects and smooth transitions.
- **GoHighLevel Integration**: Lead capture via GHL Form Iframe.
- **GA4 Tracking**: Event-based tracking for key user interactions.
- **Deployment Ready**: Out-of-the-box support for Vercel and Netlify.

## 🛠️ Local Development
1. Clone the repository.
2. Open `index.html` in any modern web browser (or use VS Code Live Server).
3. No build step required (Vanilla HTML/CSS/JS).

## 🌍 Deployment
### Vercel
- Connect your GitHub repo to Vercel.
- The `vercel.json` file will automatically handle the routing.

### Netlify
- Drag and drop the project folder into the Netlify dashboard.
- Or connect via GitHub (the `netlify.toml` file will handle configuration).

## ⚙️ Configuration
Replace the following placeholders in `index.html`:
- `YOUR_GHL_FORM_ID_HERE`: Your unique GHL form ID.
- `[Email Placeholder]`: Your contact email.
- `[WhatsApp Placeholder]`: Your business WhatsApp number.
- `[Registration Number Placeholder]`: Your MARA registration number.
